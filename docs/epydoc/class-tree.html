Class Hierarchy

  • antaresia.management.elsewhere.actions.ActionClasses: Used as a lookup for action names and which classes they map to.
  • antaresia.management.elsewhere.support.DeploymentHelper: This is the 'helper' property on the Deployment class which contains a bunch of shortcut methods for accessing different pieces of the deployments configuration.
  • antaresia.management.elsewhere.ssh.SSHSession: Pxssh interaction wrapper.
  • object: The most base type
    • antaresia.management.elsewhere.actions.ActionErrors: Re-usable error messages that are common to some actions.
    • antaresia.management.elsewhere.actions.ActionRunner: Runs an action list from a deployment instruction set.
    • antaresia.management.elsewhere.apache.ApacheErrors: Common apache errors.
    • antaresia.management.elsewhere.base.BaseAction: Base class for all actions
      • antaresia.management.elsewhere.apache.BaseApacheAction: Base apache action that contains re-usable methods.
        • antaresia.management.elsewhere.apache.ApacheRestartAction
        • antaresia.management.elsewhere.apache.ApacheStartAction
        • antaresia.management.elsewhere.apache.ApacheStopAction
      • antaresia.management.elsewhere.scm.GitUpdateAction: GIT repo udpate action
      • antaresia.management.elsewhere.scp.SCPPushAction: The SCP Push action pushes files up to a server through scp.
      • antaresia.management.elsewhere.messaging.StdoutAction: Action that prints out the final messages from support.Deployment.messages to stdout.
    • antaresia.management.elsewhere.base.BaseActionMeta: This is the base class for the meta property on the BaseAction class.
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • antaresia.management.elsewhere.exceptions.ActionCreationError
        • antaresia.management.elsewhere.exceptions.ActionError
        • antaresia.management.elsewhere.exceptions.ActionNotAvailable
        • antaresia.management.elsewhere.exceptions.ActionRequirementsError
        • antaresia.management.elsewhere.exceptions.SSHErrorDetected
        • antaresia.management.elsewhere.exceptions.TransactionRevertError
    • antaresia.management.elsewhere.support.Clients: A simple clients wrapper object that is attached to deployment.clients.
    • antaresia.management.elsewhere.support.Deployment: The Deployment class acts much like the request object does in frameworks like django or rails.
    • antaresia.management.elsewhere.messaging.ExceptionMessage: Every exception message stored in the Messaging helper support.Messaging should be an instance of this class.
    • antaresia.management.elsewhere.scm.GitErrors: Wraps up git error messages
    • antaresia.management.elsewhere.support.Messages: This is a messaging helper that is part of the deployment class.
    • antaresia.management.elsewhere.svn.SVNRepo: SVN repository command wrapper.
